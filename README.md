# 📊 Power BI Ecommerce Sales Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Ecommerce Sales data. The dashboard helps visualize sales performance, profit, quantity sold, customer behavior, payment methods, and state-wise sales using interactive charts and slicers.

---

## 🎯 Objective

The objective of this project is to transform raw Ecommerce sales data into meaningful business insights using Power BI.

---

## 🛠 Tools Used

- Power BI Desktop
- Power Query
- DAX
- CSV Dataset

---

## 📂 Dataset

The project uses two CSV files:

- Orders.csv
- Details.csv

---

## 📊 Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold
- Average Order Value (AOV)
- Sales by State
- Quantity by Category
- Profit by Month
- Quantity by Payment Mode
- Top Customers
- Profit by Sub-Category
- Quarter Filter
- State Filter

---

## 📈 Key Insights

- Maharashtra generated the highest sales.
- Clothing contributed the highest quantity.
- COD was the most preferred payment mode.
- Saree generated the highest profit among all sub-categories.
